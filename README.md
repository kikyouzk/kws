# KOS Web Shell
- The operating terminal for the KOS industrial real-time operating system.

# About KOS
- KOS is a cross-platform industrial real-time operating system that can run directly on bare chips or inside host environments such as Windows and Linux.
- KOS is a powerful application platform that provides a file system, security infrastructure, communication infrastructure, and the core libraries and frameworks needed by applications.
- In addition to the application platform, KOS also provides a Controller subsystem and an IoT Device subsystem for controller and IoT device platforms.
